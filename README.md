### Startup

#### Server

1. Initialize node

```
cd server
npm i
```

2. Add a `.env.prod` or `.env.dev`
3. Build

```
npm run build
npx prisma generate
```

4. Run the app

```
npm run prod/dev
```

#### Application

1. Gradle Sync and Build
2. Run the application. The app was designed with Pixel 4a, but it works with other models.
